# 🌸 Task 1 — Iris Flower Classification Dashboard

### Machine Learning Based Flower Species Prediction System

An interactive dashboard that predicts flower species using measurements of petals and sepals.

### Features

✅ Flower Species Prediction  
✅ Exploratory Data Analysis  
✅ Correlation Heatmaps  
✅ Interactive Charts  
✅ Real-Time Prediction  
✅ Random Forest Classification

### Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Plotly
- Joblib

### Species Predicted

🌸 Iris-setosa  
🌸 Iris-versicolor  
🌸 Iris-virginica

# 📊 Task 2 — Unemployment Analysis Dashboard

### Covid-19 Impact Analysis Using Machine Learning

A dashboard analyzing unemployment trends across India during Covid-19.

### Features

✅ State-wise Analysis  
✅ Urban vs Rural Comparison  
✅ Trend Visualization  
✅ Correlation Heatmap  
✅ Future Prediction System  
✅ Interactive Dashboard

### ML Model

- Linear Regression

### Key Insights

📈 Covid-19 increased unemployment significantly  
📈 Recovery differed across states  
📈 Labour participation strongly impacts unemployment

A predictive dashboard estimating used car prices using vehicle features.

### Features

✅ Car Resale Prediction  
✅ Price Distribution Analysis  
✅ Heatmaps  
✅ Data Insights  
✅ Interactive Dashboard

### ML Model

### Key Insights

# 💻 Installation & Setup

Clone repository:

```bash
git clone https://github.com/panthinipatel5/OIBSIP.git
```

Move into folder:

```bash
cd OIBSIP
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run any dashboard:

```bash
streamlit run app.py
```
