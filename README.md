# 💻 Installation & Setup

Clone repository:

```bash
git clone https://github.com/panthinipatel5/OIBSIP.git
```

Move into folder:

```bash
cd OIBSIP
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run any dashboard:

```bash
streamlit run app.py
```
