# 🌸 Iris Flower Classification Dashboard

## Intelligent Flower Species Prediction using Machine Learning

An interactive Machine Learning project developed using Python and Streamlit to classify Iris flowers into species using petal and sepal measurements.

This project combines Exploratory Data Analysis (EDA), visual storytelling, and Machine Learning to provide real-time predictions through a user-friendly dashboard.

---

## 🚀 Features

✅ Real-time Flower Prediction  
✅ Interactive Streamlit Dashboard  
✅ Species Distribution Analysis  
✅ Correlation Heatmap  
✅ Scatterplots & Pairplots  
✅ Machine Learning Classification  

---

## 🌼 Species Predicted

- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

Python • Pandas • NumPy • Scikit-learn • Streamlit • Seaborn • Matplotlib • Joblib
