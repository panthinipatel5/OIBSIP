# 🎯 Project Objective

The objective of this project is to analyze unemployment trends in India during the Covid-19 pandemic and extract meaningful insights using Data Science and Machine Learning techniques.

The project aims to:

* Analyze unemployment trends across different states.
* Compare unemployment rates in urban and rural areas.
* Understand the impact of Covid-19 on employment.
* Build a predictive model for unemployment rate estimation.
* Develop an interactive dashboard for data exploration and prediction.

---

# 🌍 Problem Statement

The Covid-19 pandemic significantly affected employment opportunities across India, leading to economic uncertainty and workforce disruptions.

Understanding unemployment patterns is essential for:

* Policy makers
* Researchers
* Economists
* Business analysts

This project leverages historical unemployment data to identify trends, visualize patterns, and predict future unemployment rates using machine learning techniques.

# 📂 Dataset Information

The dataset contains unemployment statistics of Indian states during the Covid-19 pandemic.

## 📌 Dataset Features

| Feature                                 | Description                     |
| --------------------------------------- | ------------------------------- |
| Region                                  | State Name                      |
| Date                                    | Record Date                     |
| Estimated Unemployment Rate (%)         | Unemployment percentage         |
| Estimated Employed                      | Number of employed people       |
| Estimated Labour Participation Rate (%) | Labour participation percentage |
| Area                                    | Urban / Rural                   |

### Dataset Purpose

The dataset provides insights into employment conditions across India and enables analysis of economic changes caused by the pandemic.

# 📈 Key Insights

* Covid-19 significantly increased unemployment rates across India.
* Urban areas were more affected during lockdown periods.
* States like Bihar, Haryana, and Jharkhand showed higher unemployment rates.
* Employment recovery patterns differed across states.
* Labour participation rate strongly impacts unemployment trends.
